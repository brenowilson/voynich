# Voynich

Independent structural analysis of the Voynich Manuscript from primary images.

This project builds an observational, auditable and reproducible corpus without importing segmentation, glyph identity, linguistic or semantic assumptions from existing transliterations.

## Core principle

We extract structure before assigning meaning.

External literature and transliterations are preserved as isolated reference material and may be used only for post-freeze comparison. They do not define the canonical corpus.

## Scope

- primary-source image acquisition and provenance;
- independent page, region, line and glyph-instance annotation;
- explicit uncertainty and reversible decisions;
- structural, geometric, graph and spectral analysis;
- null models, perturbation tests and blind validation;
- isolated comparison against external scientific literature and transliterations.

## Repository boundary

This repository is the sole working repository for the Voynich project. The `Riemann` repository is a separate project and must not be modified as part of this work.
