# Contributing

## Required principles
- Separate observation from interpretation.
- Preserve uncertainty rather than forcing a decision.
- Keep every derived datum traceable to source coordinates and hashes.
- Do not consult external transliterations while producing canonical observational annotations.
- Treat examples as explanatory approximations unless explicitly formalized.
- Register experiments before treating them as confirmatory.

## Changes
Use focused commits. Methodological changes require an ADR. Schema changes require migration notes and version increments.

## Data
Do not commit third-party images or papers without confirming redistribution rights. Store metadata, checksums and source locations when local redistribution is not permitted.
