# Scientific Sources

This area catalogs serious scholarship and primary institutional documentation. Inclusion means the work is relevant and citable, not that every conclusion is accepted.

## Evidence classes
- `FOUNDATIONAL`: primary object documentation or broadly reliable constraints.
- `SOUND_ASSUMPTION_DEPENDENT`: serious work whose conclusions depend on a representation or model.
- `HYPOTHESIS_ONLY`: useful for comparison but not a project foundation.

Store metadata and source locations first. Commit full text only when redistribution rights permit it.
