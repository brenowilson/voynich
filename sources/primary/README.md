# Primary Sources

Metadata, manifests, institutional identifiers, rights statements and checksums for manuscript images belong here.

The uploaded 209-page HolyBooks PDF is a useful access facsimile but is not canonical for pixel-level analysis because it is watermarked, recompressed and inconsistently framed. The canonical source target is the holding institution's image set.

Do not commit large image sets until storage and redistribution rules are decided.
