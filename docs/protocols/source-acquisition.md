# Source Acquisition Protocol

1. Prefer the holding institution's highest-resolution files.
2. Record source URL, retrieval timestamp, institutional identifier, rights statement and original filename.
3. Compute SHA-256 before any transformation.
4. Preserve original bytes in read-only storage.
5. Store transformed derivatives separately with transformation parameters and parent hash.
6. Do not use watermarked or recompressed facsimiles as canonical analytical images when higher-quality institutional images exist.
