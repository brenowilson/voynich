# Terminology

- **Source image:** immutable image obtained from a documented source.
- **Folio:** manuscript-side identifier using institutional foliation where available.
- **Region:** bounded area selected by an explicit geometric rule.
- **Line candidate:** visually aligned sequence region; not assumed to be a linguistic line.
- **Glyph instance candidate:** bounded graphical unit proposed by a segmentation rule.
- **Visual class:** provisional grouping by observable form; not a letter.
- **Sequence:** ordered observations under a declared ordering rule; not necessarily a word or sentence.
- **Corpus freeze:** immutable versioned set of observations.
- **Operator:** mathematically defined transformation constructed from a declared representation.
- **External reference:** any third-party transcription, classification or interpretation.
- **Hypothesis:** replaceable interpretive or generative claim not embedded in the canonical observations.
